## Report Template Maker (ngReportMaker)

Build with AngularJs + Typescript


An angular directive that allows you to create a customized report template.

## Installation

1. ``npm install``
2. Minify resources
    * **Run** ``webpack``
3. Install typings
    * **Run** ``tsd install``
